# Disaster Preparedness and Response Education System for Schools and Colleges

A Python-based educational project designed to improve disaster awareness and emergency preparedness in schools and colleges.

## Overview

Educational institutions often lack practical disaster preparedness training for students and staff. This project provides a simple digital platform to spread awareness about natural disasters such as earthquakes, floods, and fires. It helps users understand safety procedures, emergency responses, and preventive measures.

## Features

- Disaster awareness information for common natural disasters
- Safety instructions and emergency response guidelines
- Educational content for students and staff
- Simple and easy-to-use interface
- Helps institutions promote preparedness training

## Technologies Used

- Python

## Project Objective

The goal of this project is to educate students and staff about disaster safety and improve preparedness in educational institutions.

## How It Works

1. The user opens the application.
2. The system displays available disaster categories.
3. The user selects a disaster type.
4. The application shows safety precautions, emergency steps, and response guidelines.
